# 装修花费记录系统

## 技术栈

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL

## 功能描述

该系统可以记录用户的装修花费情况，用户可以在系统中添加装修项目、记录花费以及预算等信息。
